Overview

Higher-order methods for NSArray.

These methods are inspired by Clojure. None of these methods mutate the array being called on.

any:

Returns YES if block returns YES for atleast one element in the array

- (BOOL)any:(BOOL ( ^ ) ( id obj ))block

Parameters

block

The predicate that determines the test atleast one element must pass

Discussion

Returns YES if block returns YES for atleast one element in the array

Example

 NSArray *array = @[@(1), @(-1), @(2), @(4)];
 [array any:^BOOL(id obj) {
     return [obj integerValue] < 0;
 }];
 --> YES

groupBy:

Applies block to each element and uses the return value to determine what group the element should be added to

- (NSDictionary *)groupBy:(id ( ^ ) ( id obj ))block

Parameters

block

The block whose return value determines how to group

Discussion

Applies block to each element and uses the return value to determine what group the element should be added to

Example

 NSArray *array = @[@"bob", @"mat", @"sing", @"song"];
 [array partitionBy:^id(id obj) {
    return @([obj length]);
 }];
 --> {3: ["bob", "mat"], 4: ["sing", "song"]}

 }];
 --> [2, 0]

partition:

Divides up the array into smaller arrays of size N. The final partition may be smaller than N if there aren’t enough elements

- (NSArray *)partition:(NSInteger)n

Parameters

n

The partition size, must be greater than 0

Discussion

Divides up the array into smaller arrays of size N. The final partition may be smaller than N if there aren’t enough elements

Example

 NSArray *array = @[@(1), @(2), @(3), @(4)];
 [array partition:2];
 --> [[1, 2], [3,4]]

partitionBy:

Applies block to each element in the array, each time the return value of block changes a new partition of elements is started.

- (NSArray *)partitionBy:(id ( ^ ) ( id obj ))block

Parameters

block

The block whose return value determines the partition

Discussion

Applies block to each element in the array, each time the return value of block changes a new partition of elements is started.

Example

 NSArray *array = @[@"bob", @"cat", @"mat", @"sing", @"song"];
 [array partitionBy:^id(id obj) {
     return @([obj length]);
 }];
 --> [["bob", "cat", "mat"], ["sing", "song"]];
